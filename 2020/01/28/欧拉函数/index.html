欧拉函数

欧拉函数是一个用于求1~n里面所有的与n互质数字的个数

欧拉函数的性质

公式 解释
$φ(x y) = φ(x) φ(y)$ 当且(x,y)==1
$φ(x) = x - 1$ 当且仅当x为质数时
$φ(n) = (p-1) * p^{k-1}$
$φ(n)=n∗∏(1−1/p_i)$ pi是n的质因子
$a^{φ(m)}==1%m$ 当且仅当(a,m)==1
$s = n * (φ(n)/2)$ s：小于n且与n互质的数之和
$φ(n p)=φ(n) p$ 当p为质数且n%p==0
$φ(n p)=φ(n) (p-1)$ 当p为质数且n%p!=0
